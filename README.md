# Permitted

> Verified Rules & Decisions for International Students

**Permitted** provides clear, verified, consequence-aware answers to high-risk questions international students face in Canada. The product prioritizes certainty, correctness, security, and trust over speed or coverage.

## Tech Stack

- **Frontend:** React (Vite)
- **Backend:** Node.js + Express
- **Database:** PostgreSQL + Prisma
- **Hosting:** AWS EC2 + Nginx

## Getting Started

### Prerequisites

- Node.js v20+
- PostgreSQL 15+

### Installation

```bash
# Clone the repository
git clone https://github.com/your-username/permitted.git
cd permitted

# Install dependencies
npm install

# Start development servers
npm run dev
```
